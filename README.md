# 肖子乐的个人网站

这是我的个人网站 / 博客，基于 Beautiful Jekyll 主题构建。这里会发布技术笔记、读书记录与个人随笔。

## 访问
站点地址：https://xzl80.github.io

## 本地运行
1. 安装 Ruby、Bundler 和 Jekyll（参考 Jekyll 官方文档）
2. 在仓库根目录运行：
   bundle install
   bundle exec jekyll serve
3. 在浏览器访问 http://localhost:4000

## 修改站点内容
- 文章放在 `_posts/`，页面放在根目录（例如 `aboutme.md`）。
- 配置请编辑 `_config.yml`，修改站点标题、作者、社交链接等。

## 部署
本仓库通过 GitHub Pages 发布。请在仓库 Settings -> Pages 中确认分支和路径（通常是 main / gh-pages / root）。
